# CHANGELOG

## [2.27.0-L2.0.0] - 2026-07-21

Второй публичный релиз после L1, основанный на WorldPainter 2.27.0.

### Добавлено

- Пять локализаций: немецкая, французская, испанская, итальянская и упрощённая китайская.
- Полный паритет десяти локалей во всех девяти семействах ресурсов.
- Пользовательский слой **[BETA] Система пещер** (`CaveSystem`) с трёхмерным полем плотности, крупными шумовыми залами, cheese/grand-полостями, spaghetti/backbone/noodle-тоннелями, выходами, аквиферами, пышными и натёчными областями.
- Пользовательский слой **[BETA] Айсберги**.
- Терейны `ICE` и `PACKED_ICE`.
- Команда сохранения независимой совместимой копии мира для оригинального WorldPainter 2.27.0.
- Темы FlatLaf Cyan light и FlatLaf One Dark с отдельными наборами интерфейсных иконок.
- 1107 иконок блоков/материалов и 95 sprite-иконок биомов.
- Современный Windows `IFileDialog` с fallback на Swing.
- Автоматическая и ручная проверка обновлений форка.
- Windows file association для `.world` и новый packaging toolkit для Portable, WiX/MSI и Inno Setup.
- Документация сборки и локализации.

### Изменено

- Полностью расширено покрытие локализации UI, редких диалогов, инструментов и progress-сообщений.
- Терминология блоков, растений и биомов сверена с Minecraft и локализованными Minecraft Wiki.
- 150 хэш-ключей локализации переименованы в осмысленные стабильные имена.
- FlatLaf Cyan light стала темой по умолчанию; старые удалённые темы безопасно мигрируют через compatibility aliases.
- Иконки слоёв, операций, панелей и JIDE controls стали theme-aware.
- Windows packaging вынесен из runtime-дерева в `tools/windows-packaging/`.
- Обновлены README, NOTICE, copyright headers и release metadata.

### Исправлено

- JIDE docking tooltip, цвета и динамически пересоздаваемые элементы.
- Ошибки компиляции и startup crash, найденные во время очистки тем.
- Повреждённые локализованные строки, MessageFormat placeholders, escape-последовательности и HTML.
- Зависимость игровых иконок от локализованных display names.
- Большие пустые области и чрезмерная плотность в последовательных итерациях генератора пещер.

### Совместимость

- Миры без fork-only слоёв совместимы с оригинальным WorldPainter 2.27.0.
- Для миров с `CaveSystem`, `Icebergs`, `ICE` или `PACKED_ICE` добавлена отдельная команда сохранения совместимой копии.
- Deprecated enum-значения сохранены для чтения старых конфигураций.

Полный список изменений и проверенный файловый diff: [docs/CHANGES_L1_TO_L2_RU.md](docs/CHANGES_L1_TO_L2_RU.md).

## [2.27.0-L1] - 2026-07-11

Первый публичный релиз **WorldPainter Languages** на базе WorldPainter 2.27.0.

### Добавлено

- Украинская (`uk`), белорусская (`be`) и казахская (`kk`) локализации в полном паритете с русской.
- Разделение переводов на доменные bundle-файлы: strings, blocks, gamedata, layers, swing.
- Маршрутизация ключей по префиксу в WPI18n с fallback на основной bundle.
- Просьба поставить звёздочку репозиторию в диалоге поддержки.

### Изменено

- Настройки хранятся в отдельной папке `WorldPainter Languages`; при первом запуске существующие настройки копируются из старой папки `WorldPainter`, не изменяя оригинал.
- Проект переименован в WorldPainter Languages.
- Диалог поддержки обновлён под новый бренд.
- `.properties` приведены к UTF-8 без `\\uXXXX`-эскейпов.
- Выполнена вычитка RU/UK/BE/KK.

### Удалено

- 195 неиспользуемых ключей локализации, синхронно во всех языках.
- Внутренние инструменты и черновые документы из release-дерева.
