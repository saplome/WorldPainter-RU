# ROADMAP

## Цель проекта

Создать современный русскоязычный форк WorldPainter с полной совместимостью с оригиналом, стабильной сборкой и понятным развитием.

## Phase 1 - Русификация

Статус: выполняется.

Задачи:

- перевести основные окна, меню и панели;
- перевести диалоги, предупреждения, ошибки и progress-сообщения;
- перевести import/export/merge/startup/recovery сценарии;
- сохранить совместимость с оригинальным WorldPainter;
- не менять архитектуру, бизнес-логику и форматы файлов.

## Phase 2 - Архитектура локализации

Статус: запланировано.

Задачи:

- привести ключи локализации к более поддерживаемой структуре;
- убрать хрупкие строковые фрагменты и конкатенацию;
- централизовать обработку стандартных Swing-строк;
- упростить добавление новых языков;
- добавить инструменты проверки полноты перевода.

## Phase 3 - Modern UI

Статус: запланировано.

Задачи:

- внедрить современный Look and Feel;
- добавить light/dark themes;
- подготовить стиль, близкий к Windows 11;
- обновить визуальную систему без потери совместимости.

## Phase 4 - Новые возможности

Статус: запланировано.

Возможные направления:

- улучшения workflow для русскоязычных пользователей;
- дополнительные инструменты экспорта и проверки;
- улучшенная документация;
- расширенные шаблоны и примеры.

## Принцип разработки

Стабильность важнее скорости. Любое изменение должно быть минимальным, проверяемым и совместимым с исходным WorldPainter.
