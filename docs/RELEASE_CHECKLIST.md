# RELEASE CHECKLIST

Чеклист для публикации WorldPainter RU 2.27.0-r1.

## Перед сборкой

- [ ] Проверить, что работа ведётся в корне `WorldPainter-2.27.0-RU`.
- [ ] Проверить, что `.gitignore` исключает `target/`, архивы, installer-файлы, IDE-файлы и локальные логи.
- [ ] Проверить, что в репозиторий не попадают `.idea/`, `.vscode/`.
- [ ] Проверить, что в репозиторий не попадают `*.jar`, `*.exe`, `*.msi`, `*.zip`, `*.rar`, `*.7z`.
- [ ] Проверить наличие лицензии GPLv3 в публикуемом репозитории/архиве.

## Сборка

- [ ] Выполнить `mvn clean package`.
- [ ] Убедиться, что сборка завершилась успешно.
- [ ] Выполнить `mvn -pl WPGUI exec:exec`.
- [ ] Проверить, что GUI запускается.

## Ручная проверка GUI

- [ ] Главное окно.
- [ ] Главное меню.
- [ ] Toolbar и status bar.
- [ ] Диалоги открытия/сохранения.
- [ ] Создание нового мира.
- [ ] Сохранение мира.
- [ ] Закрытие мира с несохранёнными изменениями.
- [ ] Import.
- [ ] Export.
- [ ] Merge.
- [ ] Preferences.
- [ ] About.
- [ ] Ошибки и предупреждения.
- [ ] Progress dialogs.
- [ ] JFileChooser/JOptionPane.

## Документация

- [ ] Обновить README.
- [ ] Обновить CHANGELOG.
- [ ] Проверить ROADMAP.
- [ ] Проверить BUILDING_RU.

## Packaging

- [ ] Проверить наличие `java`.

- [ ] Проверить наличие WiX `candle.exe` и `light.exe`.
