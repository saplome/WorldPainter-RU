# WorldPainter Languages 2.27.0-L2.0.0

Неофициальный многоязычный форк WorldPainter 2.27.0.

## Главное

- единый пользовательский слой **[BETA] Система пещер** (`CaveSystem`);
- крупные шумовые полости, spaghetti/backbone/noodle-тоннели и естественные выходы;
- аквиферы, лавовые области, пышные и натёчные пещеры;
- глиняные бассейны и крупные натёчные образования;
- семь вкладок детальных настроек;
- английский интерфейс и девять полных локализаций;
- темы FlatLaf Cyan light и FlatLaf One Dark;
- Windows installer и Portable со встроенной Java-средой.

## Совместимость

Обычные миры без возможностей форка совместимы с оригинальным WorldPainter 2.27.0. Для миров с `CaveSystem`, `Icebergs`, `ICE` или `PACKED_ICE` используйте команду **«Сохранить копию для оригинального WorldPainter…»**.

Внутренний ID слоя: `org.pepsoft.CaveSystem`. Метка `[BETA]` является только частью отображаемого имени и не меняет совместимость или сериализацию.

> Это черновик релиза. Перед публикацией проверьте название, описание, список файлов, запуск Setup и Portable, создание мира и экспорт карты.
